JSFoo Pune Geekup

This event is in Pune. Are you looking for the event in Udupi or Chennai?

Dhananjay Nene - JavaScript: The differentiating parts

Dhanajay Nene is a passionate programmer. He is currently Chief Architect at Vayana. Dhananjay uses python and scala at work, but is interested in programming as a craft and dabbles with many languages. He wrote his second web application using server side JavaScript in 1996, the first one used awk over CGI.

Starting with the history, Dhananjay will relate JavaScript to the overall language ecosystem and its increasing popularity in recent years. Dhananjay will also discuss the characteristics of JavaScript language beyond what is immediately obvious in simple web JavaScript tutorials.

Rakesh Pai: Asynchronous Patterns, Particularly in Node.js

Rakesh Pai is the founder and Chief Do-It-All at Errorception. He tames JavaScript by day, and night too. He is currently fascinated by robots and lasers. Interestingly, Rakesh hasn't been successful yet in building a robot that can write JS code and that successfully passes JSHint. The robot disagrees about semicolon usage. It's all very stressful.

Rakesh will demonstrate how to wield the event loop LIKE A BOSS! His talk will go over the history of asynchronous patterns in JavaScript, and how the rise of node has given us new ways to work with event loops. Rakesh's talk will contain obligatory mentions of callback hell.
