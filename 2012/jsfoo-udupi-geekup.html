JSFoo Udupi Geekup

This event is in Udupi. Are you looking for the event in Pune or Chennai?

Sunil Pai: Why JavaScript wins

Sunil Pai is a Tech lead on the Maps team at Yahoo!. Originally from Manipal, he has worked in a number of environments including design firms, startups, government and MNCs. Sunil is especially interested in developer tools, user interfaces, and really nice guitars.

In his presentation, Sunil Pai will talk about the history of JavaScript, its benefits, the environments in which the language is used and deployed, and why it has become a cultural phenomenon.

Dr. Devi Prasad: The Design and Implementation of CSS3 Animations

Devi Prasad is Senior R&D Manager at Robosoft Technologies. For the past one year, he has been studying the design and implementation of two popular Web browsers: Google Chrome and Mozilla Forefox. He is interested in the multi-process architecture of Chrome, optimization techniques of the V8 JavaScript engine, and Chrome's Renderer architecture. In the last few months he has documented the design and implementation details of the CSS parers and rendering stages of Chrome and Firefox.

Following from Sunil Pai's talk, Devi Prasad will deep dive into the use of CSS3 animations for modern Web apps. He will explain the distinctions and similarities between JavaScript driven animations and CSS animations, covering the best practices for using animations in Web apps. Finally, Devi Prasad will outline the implementation details of CSS3 animations in Chrome and Firefox.

Sidharth Kuruvila: Writing good javascripts

Sidharth Kuruvila knows JavaScript and has written a few lines of code in that language. Sidharth likes long walks in the moonlight, and turtles.

Sidharth Kuruvila will round up the day by discussing some tips for writing reliable JavaScript. His talk will act as an introduction to unit testing with Jasmine.
